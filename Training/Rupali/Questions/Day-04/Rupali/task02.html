Types of HTML and CSS format

  1. Inline
  2. Internal
  3. External


Inline CSS

Inline CSSis when we type css attribute and value in tag attribute itself, ie. style atribute


Internal CSS

Internal CSSis when we type css attribute and value in style tag inside style tag in Head section


External CSS

External CSS is when we type external css attribute and value in an external stylesheet and add to the page usding link tag in Head tag.
