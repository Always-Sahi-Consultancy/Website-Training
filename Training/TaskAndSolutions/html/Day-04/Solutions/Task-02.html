HTML CSS

Types of HTML CSS format are:

  1. Inline
  2. Internal
  3. External

Inline CSS

Inline Css is when we type the css atributes and values in the tag attibute itself, ie. in style attribue


Internal CSS

Internal Css is when we type the css attributes and value in style tag inside style tag in Head section.


External CSS

External Css is when we type the css attributes and value in an external stylesheet and add to the page using link tag in the head tag.
